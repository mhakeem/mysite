This is a README file for mysite.

This repo has the complete implementation files for the Django 1.4 framework tutorials.

You do not need to define or type absolute paths for your folders, files, and database.
The settings.py file includes a line of code that makes the project portable and work on 
any machine.

For your convenience, please delete sqlite.db and type syncdb in your command line:
python manage.py syncdb
You will be able to create a new database file and add your super user and data. 

Please visit https://docs.djangoproject.com/en/1.4/ for more information on Django API.
